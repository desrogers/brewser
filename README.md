# brewser
simple web app that helps users keep track of brews they have and haven't tried, whether it's coffee, tea, or beer.

## Structure
```
.
├── README.md
├── config
│   ├── connection.js
│   └── orm.js
├── controllers
│   └── brewsController.js
├── db
│   ├── schema.sql
│   └── seeds.sql
├── models
│   └── brew.js
├── package.json
├── public
│   └── assets
│       ├── css
│       │   └── style.css
│       ├── img
│       │   └── flight.png
│       └── js
│           └── app.js
├── server.js
├── tree.md
└── views
    ├── index.handlebars
    ├── layouts
    │   └── main.handlebars
    └── partials
        └── brews
            └── brews-block.handlebars
```
